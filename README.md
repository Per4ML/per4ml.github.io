## Per4ML Research Group
- This repo contains the source of the github.io website for **Per4ML**

(The Github Pages website is generated by a push to the default branch)
___
### Static Site Generation
Static html files are generated from `*.in` files
using information in `./contents/`

Static file generation can be accomplished by using the `./deploy.sh` script
- No arguments are needed for development/local file creation

*For normal updates (git push),
generation is automated in `./.github/workflows/deploy.yml`*
___
### Content Files
Files within `./contents/` contain information to complete `*.in` files

Content filenames are the corresponding `*.in` file's prefix,
and include example usage

*Follow usage of examples or other entries*

*Files are parsed by `updatehtml.py`
___
### updatehtml.py
This script finds `*.in` files,
uses the contents of files within `./contents/`,
and generates static files to be served.

This script contains any specific parsing logic
